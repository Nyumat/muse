Introduction

Muse is an audio-based streaming platform which allows users to upload songs, create playlists, and share music with others. This API provides endpoints to interact with songs, playlists, and user profiles, enabling developers to build rich applications on top of the Muse platform.

Explore the API

API Docs

Authentication

Each of the API endpoints require authentication using JWT tokens. Obtain a token by registering and logging in through the authentication endpoints.

Endpoints

// Songs
GET /api/songs
GET /api/songs/{id}
POST /api/songs
PUT /api/songs/{id}
DELETE /api/songs/{id}

// Playlists
GET /api/playlists
GET /api/playlists/{id}
POST /api/playlists
PUT /api/playlists/{id}
DELETE /api/playlists/{id}

// Users
GET /api/users
GET /api/users/{id}
POST /api/users
PUT /api/users/{id}
DELETE /api/users/{id}
